# MKMD Manga Reader 📚

Premium manga reader with MangaDex API. Dark UI, smooth UX, no ads.

## Features
- 🔍 Search 100k+ manga
- 📖 Vertical/Horizontal reader
- 💾 Bookmark progress
- ⌨️ Keyboard navigation
- 🎨 2 themes
- 📱 Mobile responsive
- ⚡ Lazy load pages

## Deploy
1. Fork repo
2. Settings → Pages → Enable
3. Done: `https://username.github.io/mkmd-manga-reader/`

## API
Powered by [MangaDex](https://api.mangadex.org/docs/) - Free, legal, rate-limited to 5 req/s

## Tech
HTML5 + CSS3 + Vanilla JS + MangaDex API
